How to Add Face Images
  1. Create a folder with the student's name in the dataset/ directory
  2. Add multiple clear face images (JPG/PNG format) of the student
  3. Ensure good lighting and clear visibility of the face
  4. Images should be from different angles for better recognition
  5. Recommended: 5-10 images per student
